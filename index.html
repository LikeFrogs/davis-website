Phantom Frame (Working Title)

First-Person Survival Horror Game

Phantom Frame is a first-person survival horror game in which the player uses an enchanted polaroid camera to photograph otherwise invisible monsters. The game takes place in a noir-inspired setting in which our hero, a freelance detective, finds herself down on her luck, until she stumbles across a puzzling case, and a strange polaroid camera that can photograph the unseen. Dead-set on following this thread of crime, she pursues dangerous beings which can only be seen with this strange camera, and slowly unravels the mystery behind the string of strange supernatural stuff that's going on in town.

    My contributions:
  • Created the AI behaviors for the monsters
  • Wrote general narrative and demo level scenario

Gungi/War Council

Abstract Strategy Game in Three Dimensions

This is a short-term project I started over the summer as a chance to work on something a little different, with the hopes of publishing on mobile sometime soon. The game is similar to chess and shogi (Japanese chess), but with the ability to stack pieces on top of each other. It is based on a game appearing in the popular anime Hunter X Hunter with rules developed on Reddit that I modified slightly after playtesting. This is also a solo project with all art and programming by me. The game is nearly done, so expect to see a build go live soon! The coolest features I've implemented are:

  • An event-based system for moving pieces on your turn
  • Calculating all legal moves for the current player (and thus determining if they are in checkmate)

Project Rose Garden

Turn-based Stealth Strategy Game

Project Rose Garden was a somewhat longer term project that I used primarily as a way to learn Unity 3D. Rather than use plugins or pre-writen algorithms, I implemented all the features myself. I tackled many challenges while working on this project and learned a lot! I have since moved on to other projects, but working on this game was invaluable in developing my skills early in my career.

    The most important features I worked on are the following:
  • A custom variant of A* enemy pathfinding
  • A dynamic UI dialogue system
  • Combat, patrol and search behaviors for NPCs

Stealth Cooking

In this game the player takes control of a hungry teen who gets up in the middle of the night and must make pizza without waking the house. Click the image to play the game on Itch!

Entangled

3D Puzzle-Platformer

This very short-term team project was basically a tech demothat we put together to test our skills. The game's core mechanic is the player's ability to control two instances of himself that can both interact with different parts of the world, but not each other. The code is available on github.

    The features I've worked on were:
  • The handling of the two instance system: death and respawning of one or both characters
  • Moveable blocks for planned block puzzles
